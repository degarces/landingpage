This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

## Project Hierarchy Reference

```
my-next-app/
|-- README.md - Project overview plus current file hierarchy reference.
|-- package.json - App metadata, scripts, and dependency list.
|-- package-lock.json - Locked dependency tree generated by npm install.
|-- next.config.mjs - Next.js configuration placeholder (currently default).
|-- postcss.config.mjs - PostCSS pipeline that wires Tailwind processing.
|-- eslint.config.mjs - ESLint flat config extending Next core rules.
|-- components.json - shadcn/ui registry config for component scaffolding.
|-- jsconfig.json - Editor path aliases mapping `@/*` to `src/*`.
|-- tsconfig.json - TypeScript settings powering IDE tooling and incremental builds.
|-- .gitignore - Git exclusion list for build output, env files, and artifacts.
|-- To-Do/
|   `-- 003-Desktopnav - Placeholder task stub for upcoming desktop nav work.
|-- Done/
|   |-- 001-Larger Logo - Completed plan documenting the navbar logo resize.
|   `-- 002-Mobilenav-Title - Completed plan detailing the mobile title update.
|-- public/
|   |-- logo.svg - Primary logomark displayed in the navbar.
|   |-- globe.svg - Globe-themed illustration asset.
|   |-- window.svg - Window UI illustration asset.
|   |-- file.svg - File/document illustration asset.
|   |-- next.svg - Default Next.js logo asset.
|   `-- vercel.svg - Default Vercel logo asset.
`-- src/
    |-- app/
    |   |-- layout.js - Root layout applying the system font stack, navbar, and footer.
    |   |-- page.js - Landing hero with CTAs and green “ThorneCheck” highlight.
    |   |-- globals.css - Tailwind import, theme vars, and global font stack overrides.
    |   |-- favicon.ico - Favicon served by the App Router.
    |   |-- about/
    |   |   `-- page.jsx - Static About page placeholder content.
    |   `-- pricing/
    |       `-- page.jsx - Static Pricing page placeholder content.
    |-- components/
    |   |-- Navbar.jsx - Responsive navbar with mobile title and slide-in drawer.
    |   |-- Footer.jsx - Footer with link grid, socials, and current-year stamp.
    |   `-- ui/
    |       `-- button.jsx - Variant-capable button built with class-variance-authority.
    `-- lib/
        `-- utils.js - `cn` helper combining clsx and tailwind-merge utilities.
```

Notes: `.next/` and `node_modules/` contain build artifacts and installed dependencies and are intentionally omitted from the reference chart.
